Canonic Sequence Web App

This project combined the DOM-manipulation techniques I learned from making this project gallery with my previous, terminal-based program to make a fully interactive web app to explore 2x2 canonic sequences. It will hopefully make a nice educational resource for musicians with an interest in theory. The set of all 2x2 canonic sequences is generated algorithmically in JavaScript, and populates a column to the left-hand side of the screen. To the right, details about individual patterns the user clicks on are displayed, and above is a dynamic menu by which they can be filtered. Information about each pattern is also dynamically generated via various algorithms, and even the images were compiled from algorithmically generated LilyPond code.
