Canonic Sequence Terminal App

Pursuing this project was what originally got me into coding! I'd been working on an academic musical project examining a class of musical object known as canonic sequences. This was a tool I wrote in the C language to generate a specific subset of these patterns and process them, and create formatted LaTeX files containing data on each pattern along with musical examples algorithmically typeset in LilyPond.
